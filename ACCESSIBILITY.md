# دسترس‌پذیری (Accessibility)

راهنمای دسترس‌پذیری وایب‌کر برای کاربران با معلولیت و ناوبری با کیبورد.

## پیاده‌سازی‌های فعلی

### ۱. Skip Link
- لینک «رفتن به محتوای اصلی» برای کاربران کیبورد
- با اولین Tab نمایش داده می‌شود
- هدف: `#main`

### ۲. Focus قابل مشاهده
- `:focus-visible` با outline بنفش برای المان‌های فوکوس‌شده
- فقط با کیبورد (نه با کلیک ماوس) نمایش داده می‌شود

### ۳. ARIA و نقش‌ها
- `aria-label` برای دکمه‌ها و لینک‌های بدون متن
- `aria-current="page"` برای آیتم فعال در ناوبری
- `aria-hidden="true"` برای آیکون‌های تزیینی
- `role="alert"` برای پیام‌های خطا
- `role="banner"` برای هدر سایت

### ۴. فرم‌ها
- `label` با `htmlFor` برای inputها
- `aria-label` برای search input

### ۵. تصاویر
- `alt` متن‌دار برای همهٔ تصاویر محتوا
- آواتارها: `alt` با نام کاربر یا «کاربر»

### ۶. ساختار صفحه
- `lang="fa"` و `dir="rtl"` در HTML
- سلسله‌مراتب heading (h1 → h2)

## چک‌لیست بهبود

- [ ] تست با screen reader (NVDA, VoiceOver)
- [ ] بررسی کنتراست رنگ با WebAIM Contrast Checker
- [ ] تست کامل ناوبری با Tab و Shift+Tab
- [ ] کاهش حرکت (prefers-reduced-motion) برای انیمیشن‌ها
