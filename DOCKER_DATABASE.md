# دیتابیس پروژه — فقط سرور جدید

## منبع دیتا و اتصال

**همهٔ داده‌ها و اتصال‌ها فقط از سرور زیر خوانده می‌شوند:**

- **Host:** `pgsql.feedban.ir`
- **Port:** `5174`
- **Database:** `wibe`
- **User:** `postgres`

هیچ اطلاعاتی از پستگرس داکر لوکال خوانده یا لود **نمی‌شود**. مقدار `DATABASE_URL` و `DIRECT_URL` در `.env` باید فقط به همین سرور اشاره کند.

---

## فایل `docker-compose.yml`

فایل `docker-compose.yml` در پروژه فقط برای **استفادهٔ اختیاری** است (مثلاً اگر کسی بخواهد یک پستگرس لوکال برای تست جداگانه بالا بیاورد). اپلیکیشن وایب به آن وصل نیست و از آن دیتا نمی‌خواند.

برای اطمینان: در `.env` فقط آدرس سرور `pgsql.feedban.ir` را نگه دار.

---

## دستورات مفید (همه روی سرور جدید)

| دستور | توضیح |
|--------|--------|
| `npm run db:studio` | Prisma Studio روی دیتابیس سرور (wibe) |
| `npx prisma db push` | به‌روزرسانی schema روی سرور |
| `npm run db:export` | خروجی گرفتن از **همین سرور** به `prisma/seed-data.json` |
| `npm run db:seed` | وارد کردن دیتا از `seed-data.json` به **همین سرور** |

---

## امنیت

- پسورد دیتابیس را فقط در `.env` قرار بده و `.env` را در git کامیت نکن.
